Busca em Largura (BFS)

A Busca em Largura (BFS) é um dos algoritmos mais conhecidos e utilizados para a exploração de grafos, explorando todos os vértices a partir de um vértice inicial em "camadas". Utiliza uma fila (FIFO) para gerenciar os vértices, garantindo que os mais próximos sejam visitados primeiro. Em grafos não ponderados, a BFS encontra o caminho mais curto entre o vértice inicial e qualquer outro vértice acessível. A complexidade do algoritmo é O(V+E), tornando-o eficiente para grafos grandes. BFS é comumente usado em problemas de busca de caminho mais curto, verificação de conectividade de grafos e resolução de problemas como "encontrar todos os caminhos de distância k" em um grafo.
