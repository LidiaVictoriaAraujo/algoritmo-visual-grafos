Busca em Profundidade (DFS)

A Busca em Profundidade (DFS) é um algoritmo fundamental para a exploração de grafos, que explora os vértices profundamente antes de voltar e explorar caminhos alternativos. Pode ser implementado de forma recursiva ou iterativa, utilizando uma pilha para gerenciar a ordem dos vértices. A DFS é vantajosa para problemas que exigem a exploração completa de um caminho, mas não garante encontrar o caminho mais curto. É amplamente utilizada para detectar ciclos, verificar a conectividade de componentes e realizar ordenação topológica de grafos acíclicos direcionados (DAGs), com complexidade O(V+E).
